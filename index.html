📝 Adding Trading Notes
  1. Open Admin panel → find the ticker → click Edit
  2. Scroll to the Trading Notes field at the bottom
  3. Type your analysis — use $TICKER to auto-highlight symbols
  4. Click Save Ticker
Notes appear in the right panel (desktop) or Notes tab (mobile).
🔄 Updating a Chart Image
  1. Annotate your chart in StockCharts
  2. Right-click the chart → Open Chart Image In New Tab
  3. Copy the URL from the address bar
  4. Admin → Edit ticker → paste into StockCharts Image URL → Save
